Principais formatações:

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag b (não semantica).

Nessa frase tenho termo em destaque usando a tag STRONG (semantica).

Itálico / êNFASE

Nessa frase temos um termo em itálico usando a tag I (não semantica).

Nessa frase temos um termo em enfase usando a tag EM (semantica).

Texto Marcado.

Podemos criar também um texto marcado usando a tag MARK.

E no outro temos outro texto marcado.

Texto grande e pequeno.

Estamos criando um texto grande e um texto pequeno.

Texto deletado.

Podemos marcar um texto excluido para indicar que ele deve ser lido, mas não considerado.

Texto inserido.

Podemos marcar un texto como inserido para dar enfase e indicar que ele foi adicionado depois (semantica).

Existe também o subinhado com a tag U (não swmantica).

Texto sobrescrito.

Para inserir coisas do tipo x20+3.
